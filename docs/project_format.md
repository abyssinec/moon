# Project Format / Формат проекта

## English
Projects are stored as versioned JSON documents with stable IDs for tracks, clips, and assets. Paths should remain relative to the project root where possible.

Top-level fields:
- `schema_version`
- `project_name`
- `sample_rate`
- `tempo`
- `assets`
- `tracks`
- `clips`
- `ui_state`

## Русский
Проекты хранятся как versioned JSON-документы со стабильными идентификаторами для треков, клипов и ассетов. Пути по возможности сохраняются относительно корня проекта.

Основные поля:
- `schema_version`
- `project_name`
- `sample_rate`
- `tempo`
- `assets`
- `tracks`
- `clips`
- `ui_state`
